재귀적 방법
- 함수가 자기 자신을 호출하여 피보나치 수열을 계산하는 방식
- 동일한 값을 여러 번 반복해서 계산하는 문제가 발생함 -> 연산 횟수 증가 및 시간 복잡도 매우 비효율적
- N이 조금만 커져도 실행 시간이 급격하게 증가

순환적 방법
- 반복문을 이용하여 이전에 계산한 값을 저장하면서 다음 값을 구하는 방식
- 한 번 계산한 값을 다시 사용하기 때문에 불필요한 중복 연산 X
- 함수 호출이 없기에 재귀보다 메모리 사용이 적고 실행 속도 매우 빠름

| 항목     | 재귀적 방법     | 반복(순환) 방법 |
| ------ | ---------- | --------- |
| 구현 난이도 | 쉬움 (직관적)   | 쉬움        |
| 실행 속도  | 매우 느림      | 매우 빠름     |
| 시간 복잡도 | O(2ⁿ)      | O(n)      |
| 메모리 사용 | 많음 (함수 호출) | 적음        |
| 중복 계산  | 많음         | 없음        |
